# Landing Page

## Overview

This repository contains the landing page for **ZeroTrace Solutions**, built with a modern React + Vite front-end structure.

## Contents

- `frontend/` - React app source code and configuration.
- `frontend/src/` - main source code including components, pages, utils, and assets.
- `frontend/public/` - static assets for the app.

## Features

- Multi-language support (i18n) via `locales/`.
- Animated UI components in `src/components`.
- Admin section with project and whitepaper management.
- Firebase integration in `src/lib/firebase.js`.

## Development

1. `cd frontend`
2. `npm install`
3. `npm run dev`

## Build

- `npm run build`
- `npm run preview`
